команда опция -аргумент

whoami - изкарва username
pwd - print working directory (директорията в която се намираме)
cd - change directory (влизане в директория)
ls или dir - list directory (покажи директории)
cd / - отиди на началната директория/root
ls -l - long list изкарва по-подробно
ls -la - показва скрития файл
cd .. - върни една директория нагоре
абсолютен 
релативен path
man [команда] - help за команди (излиза се с q)
echo [стринг] - връща стринга
mkdir [име на папка] - създай директория
touch [име на файла] - създаване на файл
nano [име на файла] - отваряш файл (ctrl + x, y, enter)
